# PROJECT_BRIEF

## 项目目标
构建基于 LangGraph 的个人知识与任务管理 agent，支持笔记、提醒、偏好、文件导入、知识检索和工具调用，并为后续更强 RAG、本地模型和企业级扩展保留清晰边界。

## 范围
- 阶段 0：项目骨架、目录、状态设计、主图草案、存储边界、调试方案、长期记忆机制
- 阶段 1：CLI、最小 graph、SQLite、Milvus Lite、文件系统、基础工具闭环、最小可运行 demo

## 成功标准
- 有可运行的最小 agent
- 有可追踪的 graph 执行路径
- 有持久化记忆和基础检索
- 有稳定的项目记忆文件机制
- 每次迭代都能更新文档、任务状态和关键决策

## 非目标
- 现阶段不做多 agent、GraphRAG、复杂评测、vLLM、企业级权限系统
- 现阶段不做复杂 UI
- 现阶段不追求高精度 RAG 或复杂 prompt 工程

## 目标用户
- 当前阶段主要服务项目作者本人，用于个人知识与任务管理

## 第一阶段交付物
- 可运行的 CLI
- LangGraph 主图
- 基础工具系统
- SQLite 和 Milvus Lite 本地存储
- 项目长期记忆文档
