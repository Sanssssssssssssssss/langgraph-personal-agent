# PROJECT_BRIEF

## 项目目标
构建基于 LangGraph 的个人知识与任务管理 agent，支持笔记、提醒、偏好、文件导入、知识检索和工具调用，并为后续更强 RAG、本地模型和企业级扩展保留清晰边界。

## 范围
- 阶段 0：项目骨架、目录、状态设计、主图草案、存储边界、调试方案、长期记忆机制
- 阶段 1：CLI、最小 graph、SQLite、Milvus Lite 兼容封装、文件系统、基础工具闭环
- 阶段 2：交互式 CLI、destructive 确认节点、retrieval metadata/filter、文件清单能力、配置化确认策略、会话历史持久化草案、graph planning 职责拆分

## 成功标准
- 有可运行的最小 agent
- 有可追踪的 graph 执行路径
- 有持久化记忆和基础检索
- 有稳定的项目记忆文件机制
- 每次迭代都能同步文档、任务状态、关键决策和版本日志

## 非目标
- 当前阶段不做多 agent、GraphRAG、复杂评测、vLLM、企业级权限系统
- 当前阶段不做复杂 UI
- 当前阶段不追求高精度 RAG 或复杂 prompt 工程

## 目标用户
- 当前阶段主要服务项目作者本人，用于个人知识与任务管理

## 当前交付物
- 可运行的 CLI 和 REPL
- LangGraph 主图与 planning 拆分
- 基础工具系统
- SQLite 和向量索引兼容封装
- 文件导入、文件清单、知识检索
- 会话历史持久化草案
- 项目长期记忆文档
